About this Project

This web application is a collaborative ICT knowledge base created to help IT teams efficiently store, organize, and share technical and operational knowledge. In professional IT environments, it is crucial to capitalize on accumulated expertise, and a well-structured knowledge base enables teams to quickly find solutions, follow best practices, and ensure continuity of services even when personnel changes occur.

The application allows users to create, modify, and consult ICT knowledge articles, each containing detailed information about encountered problems, solutions applied, and change history. Articles can be categorized by technical fields such as infrastructure, networking, software, or hardware, and can be optionally associated with ICT components like servers, databases, routers, or business applications.

Advanced features include an intuitive search engine, dynamic filtering by categories, authors, or components, and ranking articles by popularity or relevance. The system also supports exporting articles and their metadata in PDF, JSON, or CSV formats, as well as automated exports via API for integration with other tools.

User access is managed according to roles: administrators have full control, contributors can create and edit content, while readers can view articles without modifying them. This structure ensures both collaboration and data integrity within the knowledge base.

The project showcases skills in full-stack web development, scalable relational database design, organization of technical content, and implementation of customizable user roles. It also emphasizes best practices in project management, documentation, and collaborative teamwork.

Developed by Y. John, B. Anas, Z. Osman, and S. Amir – the software developers.
